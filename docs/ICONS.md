# Flyto2 Module Icons Reference

This document lists available Lucide icons for use in `@register_module()`.

**Official Lucide Icon Browser**: https://lucide.dev/icons/

---

## Usage

```python
@register_module(
    module_id='category.my_module',
    icon='Download',  # Use PascalCase icon name
    color='#3B82F6',
    # ...
)
```

## Tips

1. **Use PascalCase**: `Download` not `download`
2. **Be semantic**: Choose icons that represent the action/concept
3. **Be consistent**: Similar modules should use similar icons
4. **Check Lucide**: Browse https://lucide.dev/icons/ for all options
5. **Fallback**: Unknown icons fall back to `Box`
